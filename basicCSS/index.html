What is LaTeX?

According to Wikipedia
LaTeX is a software system for document preparation. When writing, the writer uses plain text as opposed to the formatted text found in WYSIWYG word processors like Microsoft Word, LibreOffice Writer and Apple Pages.

It's mainly used to typeset complex math using only a few lines of code!

Take for example Cauchy's integral formula

$$
    f^{(n)}(a) = \frac{n!}{2\pi i}
                 \oint_{\gamma}\frac{f(z)}{(z-a)^{n+1}}\, dz
$$
$$f^{(n)}(a) = \frac{n!}{2\pi i} \oint_{\gamma}\frac{f(z)}{(z-a)^{n+1}}\, dz$$

LaTeX allows you to focus of the content and not worry about how things look. It is also allows you add citations and cross references quite easily using packages like BibTeX.

LaTeX can also generate beautiful looking plots using Tikz and PGFPlots
